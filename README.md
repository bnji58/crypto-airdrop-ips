## 空投IP到底在卡什么？

先说一个基础概念。

**加密货币空投**（Airdrop），是区块链项目方向符合条件的钱包地址分发代币的机制。参与者通常需要完成链上交互任务、社交媒体互动，或者单纯持有特定代币，才能拿到奖励。

项目方的核心问题是：怎么识别真实用户，把一个人撸多个账号的"女巫行为"过滤掉。

IP地址是最直接的识别维度之一。项目方的反女巫系统会扫描：

- 同一IP下有多少个钱包在做交互
- 这个IP是机房地址还是真实住宅宽带
- 这个IP有没有历史风险记录

机房IP（数据中心IP）的问题在于，它的ASN信息会直接显示是"云服务商"，比如AWS、Google Cloud、Vultr这类，而不是AT&T、Comcast这样的宽带运营商。聪明一点的项目方风控系统，会直接对机房IP进行降权甚至封号处理。

所以撸空投需要的**加密货币空投IP**，核心要求只有一个：看起来像真实住在某地的普通网络用户。
